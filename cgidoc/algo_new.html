How it Works

ERPred is a threshold dependent, N-terminal, C-terminal and remaining amino acid composition and support vector machine based predictor, which predict Endoplasmic Reticular proteins. First it divide each amino acid into 3 parts, then calculate amino acid composition of each parts. After that it run SVM classify to predict protein. If SVM score is greater than threshold, the input sequence belongs to ER proteins and if SVM score is less than threshold, the input sequence belongs to Non-ER.
